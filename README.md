# Xplora – Tourism Web Application 🌍

## 📌 Description
Xplora is a tourism web application that helps users explore travel destinations.

This project also includes a machine learning model for prediction/analysis.

## 🚀 Features
- Explore tourism data
- Machine learning model (Decision Tree)
- Data processing and storage using SQLite
- Backend using Python

## 🛠 Tech Stack
- Python
- Django (if used)
- SQLite
- Machine Learning (Decision Tree)

## 📁 Project Structure
- main.py – main application
- manage.py – Django management
- db.sqlite3 – database
- decision_tree_classifier.pkl – ML model

## ▶️ How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run project:
   python manage.py runserver
